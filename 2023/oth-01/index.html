從hexo遷移到hugo 兼 近況更新

好像變得更快更星爆了。

Jun 26, 2023
閱讀時間: 2 分鐘

存活確認

嘿對，我終於想到要更新這個blog了。
距離上次寫blog已經過了超過一年了，總覺得時間過得越來越快，但卻又感覺好多事情都沒做。當初希望每半年起碼考到一個證照，結果在我拿到JLPT N2之後就停滯至今，真的不能小瞧人類的惰性。
而我這段時間做了什麼呢？突然問的話我也想不起來，只好借助科技的力量(aka記帳本)來回想了：

  • 通過 JLPT N2，之後應該會補心得
  • 把電腦能換的零件全換了，只剩機殼沒換 (如果我把機殼也換掉，那還是同一台電腦嗎)
  • 為了改冷氣管線做了規模有點大的小型室內裝潢 (錢包：)
  • 在github做了幾個小專案，拿了一些星星
  • 在公司做了幾個內部小專案，不過沒有星星
  • 考上了研究所
  • 買了新的筆電+手機+螢幕+鍵盤
  • 確診，最後有成功存活

未來可能會更忙碌了，不過希望起碼能一個月產出一篇文章，為未來做好準備(?)

網站遷移

接下來就是本文原來的主題，從Hexo遷移到Hugo的事情啦

緣起

如上一段所說，這段時間我把電腦能換的零件全換了，而免不了的遇到的電腦重灌的問題。
在重灌後所有軟體都需要重新安裝及配置，但在我嘗試著復原我的blog時，我發現我竟然只備份到文章的部分，其他的部分都需要重新設定。
既然要重構，我不如來嘗試看看其他主題或框架吧？而就成為了今天的楔子啦。

安裝

因為我使用windows環境，因此選用Chocolatey進行安裝。(Chocolatey可以在安裝Node.js時一併安裝)

1

choco install hugo-extended

其他環境下的安裝方式可以參考官方說明

配置

1

hugo new site quickstart

文章都放在content資料夾裡面，有用過Hexo的應該都滿好上手的。

1

hugo server

可以到 http://localhost:1313/ 預覽文章~

1

hugo

確認沒問題後再將生成的靜態檔案push上去就可以了~
